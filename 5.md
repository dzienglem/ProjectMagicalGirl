挂坠是个透明的小盒。挂在兔子胸前时，就像是一块勋章。微光沉淀在碎片底部，已经很不安定，随时都会熄灭。赭石慢慢从脖子上摘下挂坠，在人类注视之下轻轻放在地面，然后后退着返回兔群中间。副会长眼光追随着赭石的一举一动，脸上的表情丝毫没有舒缓。\

副会长手无寸铁，却震慑住了以军师为首的一群兔子。她提出要查看赭石的挂坠盒，后者不敢不答应。装着发光碎片的挂坠盒已经放在副会长眼前，她却并不急于取来观看。\

在她背后，朋友靠墙坐下来休息。\
\
她没被夺来的兵器的拉胯枪托捶断骨头。但是大战步枪原本就不是为立射设计，端起来开火两次足以令人身心俱损。装填好的第三发子弹原先是立刻转向，枪口直捅没回过神来的游击队长。幸而副会长及时赶到，调停了这场争端。人类和兔子互换了人质，朋友得以被释放，而受到严重惊吓的赛兰被两只兔子搀着返回了队伍后方。\
\
“哇哦，赛兰这家伙怎么这么怂，他腿都软了耶”\
\
“得了。那枪口比脸还大，换了你也得怂”\

一老一少两只林地兔交头接耳。赛兰的确哆哆嗦嗦，却还是强打精神。连带着赭石也无暇关注交出去的挂坠盒，而是充满担忧地回望队长的背影。\

“拉斯普京呢？”\
朋友不按路数，比起刚刚才和鲸对峙过的副会长，首先关注了不在场的白兔的去向。\
\
“他说他腰断了走不了路，所以我把他丢进清扫间的水桶里了”\

朋友的脸色前所未有地严峻。\
“盖严了吗？”\
\
“你尽管放心，他要能跑出来算我输”\
\
“你当时也应该直接回去。子弹虽说只有一匣，也该够你破门了。”\
\
“谁会想到用这玩意儿破门？？”\
\
“……至少也不应该一路追着鲸上来”\
\
副会长不再对对话抱有期望，进而转过眼光，被注视的赭石下意识地一激灵。\
\
“你老实交代，这个东西——这堆碎片，是你从哪儿搞来的？？”\
\
眼下的栗毛垂耳兔的抖得说不成话。圆圆的兔眼里瞬间溢满了泪水。\
\
“你倒是说点什么啊——别哭，站直了说话！”\
被逼问的小兔子原本还能勉强忍住眼泪，副会长的警告却起到了反作用。无可奈何地看了一眼怎么都拒绝说话、现在还扑扑簌簌落泪的栗毛兔，副会长坐回地面，像是自言自语，又像是在向背后的友人抱怨。\
“看来拉斯普京说的是真的，兔子确实没胆子跟人说话”\
\
讯问一筹莫展，副会长从打开的挂坠盒里拿出发光的碎片放在手心。有着能够边缘的两块碎片拼起来能形成一个蛋状的底部，比想象中的还要重——和它相比，透明挂坠盒就像塑料纸一样没有质量；摸起来的质地则像是珐琅，只是微光能从几近透明的表面渗出。\
“好吧好吧。那你告诉我，今晚你们怎么抓到那边那个人的？这总可以吧？”\

“咳咳。这个嘛，就让我来解释吧”\
队伍里的高附清了清嗓子挺身而出。\
\
“等、等等，赛兰抓人的时候咱俩不是根本没在场吗！”\
\
“这有什么办法，队长和军师都被吓傻了，现在只有我这个老兵能跟人类交涉了不是么”\
一老一少两只林地兔子边说相声边站到了副会长眼前。\
\
“拜托你交涉的时候别瞎编”\
副会长不抱希望，以一边膝盖为支点的手臂撑住了腮，垂下眼来看地上挺直了腰杆、兔耳有别人两倍长的成年林地兔。\
\
“哎哟，小姑娘，瞧你这话说的。我都没看见，可不就是只能瞎编吗？”\
“但好在今晚的运输计划需要和本队配合一致，所以赛兰他们在哪儿干了什么，我心里都有底”\
在地下边境的生活和从事运输业的经验使得高附即使应对起人类来也并不紧张。\
\
“赛兰呢，纯粹是迫不得已。”\
“时间一到，你们这个学校内部就会化为迷宫。这一点小姑娘你已经体验到了吧？在这迷宫里，底层的巴尔扎克雕像后的这道楼梯只向上下延展，是重要的垂直交通线。不过，若是没能在迷宫化开始前登上这道台阶就会迷路。人类是如此，我们也一样。\
“今天的学校里本应该在实施宵禁。原计划的出发时间里，整座学校里都不应该有人，赛兰他们能顺利从大厅到达那个楼梯口的雕像……本该是这样的。没想到，那里却出现了个人类，就是小姑娘你被抓住的朋友啦。”\
“我跟你对天发誓，赛兰虽然脑子缺根筋，但绝对没傻到主动跟人类挑起冲突。要不是你的朋友待在那儿不肯让路，他也不会铤而走险。说实话我直到刚才都纳闷儿他究竟是怎么俘获她的——那些个豆子机炮除了打打地下人，对人类来说应该不算什么威胁才对。不过，”\
目光越过副会长，高附瞄了一眼远处坐在墙下休息的人类。高窗投下的光不够明晰，人类身体的大半陷入阴影之中。被她夺走的大战步枪倚墙而放，退掉的子弹和空了一半的弹夹则放在手边不远处的地板上。\
“——你的朋友，敢情也是将计就计啊。”\
\
“你说这都是为了登上台阶……然后呢，你们登上台阶要去哪儿?”\
\
“这还不明白吗？当然是去猎鲸啦。”\
\
……副会长皱起了眉。眼前的砂色毛球的确比被她安置进楼下水桶里的拉斯普京看起来更加体态健硕，但要上称约，大概肉也多不了几两。在他背后，组成游击队的兔子也不过几十只。相比而言，鲸——即使是刚才只有牛那么大的幼鲸，发起怒来也足够把他们踏得全军覆没。\
\
“……就凭你们……不是，为啥要猎鲸？？”\
\
“啊呀。哪里有那么多为什么，你是‘绿色和平’吗？”\
高附挠了挠耳朵。“林地兔一族自古以来就有和平利用鲸的文化……虽说我们那个屯本身已经因为鲸瘟爆发灰飞烟灭了，所以可能也就没什么文化了……赛兰的先祖是游骑兵，地下人管他们叫‘蛮夷’驱逐到地面上。如果不猎鲸，他们连生存空间都受到威胁。就连地下人不也是跟鲸有不共戴天之仇吗？还有你们，鲸可是靠人类灵魂活着的，我们猎鲸可也算是在帮你们的忙，算是做好事不留名吧”\
\
“要是你们今晚没劫持人类，这话说不定我就信了呢？”\
\
“哎哟，我不都说了吗，全是误会。再说，好像被坑了一道的是我们吧？赛兰吓得差点背过去，还被你们缴了械”高附耸了耸肩。\
\
副会长从鼻子里哼出一声。\
“那还不是因为你们昏了头，麻烦找到了人类身上。”\
\
“这我倒是……无话可说。说实话，我刚才要是在场，拼了命也会拦住赛兰做这种傻事”\
“……但你看，你们也不该大晚上的到这儿来。另一方面，猎鲸活动对人类而言也是百利而无一害，我们应该握手言和才对啊。”\
\
“能不能握手等会儿再说吧。我到这儿来，有正儿八经的原因。要不然，谁会想在这种夜里到处瞎逛”\
“我是在找这些碎片。”\
副会长抬眼一看，刚才把挂坠盒挂在胸前的小兔子仍然眼泪汪汪，合不拢的三瓣嘴看起来还是没法吐出一个字。\
\
“……不不不，这也还是我来解释吧”注意到副会长表现出的露骨不信感，高附赶紧打圆场。“赭石他不太擅长跟人类说话”\
\
“你倒是挺擅长的啊，嗯？”\
\
“咳，跑运输嘛。要吃这碗饭免不了跟各路人打交道，容不下社恐……但你别担心。如果你想知道这些碎片是啥，我帮不上忙。但要是只想知道它的来历，那我可是身临其境，看得清清楚楚”\
\
“嗯？你们不知道这些碎片是什么，为什么还要收集它？”\
\
“唔，这事儿我没法跟你解释清。啊不，不不不，不是我想隐瞒什么。”高附连连摇头。“赭石觉得这些碎片有点儿蹊跷，而它出现的地方又偏偏是猎场，所以想以后回村的时候顺便拿给把式看看。哦对了，你还不知道什么是猎场呢——”\
\
高附特意把话说得意味深长，而这丝毫不能打动副会长。\
“不就是西塔楼1203房间吗？”\
\
“呃，怎么你原来知道啊？”\
\
“我不是说了我在找东西嘛。多亏你们悄不作声地从房间里拿走了它，害得我扑空。”\
不自觉地面呈凶相的副会长把远远站在一边的赭石吓得一激灵。\
“问题是，你们捡到碎片是在什么时候，在那个房间里又看到了啥？”\
\
“看到了啥？呃……你也知道在那种迷宫里并不可能真的看到什么吧？但是捡到碎片的晚上确实啥也没发生。那天我们是去追之前跟丢了的契丹鲸的，但它并没在那个房间里出现。跟你说实话，那头鲸逃得无影无踪，直到今天，都没能找到它。”\
\
“没找到？可那个房间里现在就躺着个开膛破肚的鲸尸，你不会心里没数吧？”\
\
“啊呀，那是另一头了。总得有个候补猎物，不能指望着一头鲸过日子吧？这天眼看就要变，下雪之前要是没把鲸脂鲸皮啥的储备好，你让我们怎么点灯、怎么造胰子，游击队的弟兄拿什么补裤子”\

“……裤子？？”\
\
“对啊。要是没有条鲸鬃大氅，壕沟里的冷气能把你冻成条棍子。原本今晚我们还得回收那个房间里的尸体，要不就不赶趟了。话说回来，无论你们是何方神圣，那个尸体留在那儿都只会找麻烦，得尽快收拾是不是？在这一点上咱们立场一致吧？”\
\
“行了，你说的猎场我懂了。那就说说碎片的事儿吧。它有什么蹊跷的？”\
\
“这是地下人的东西，人类造不出来。但是地下人可不会到地上来。无论这东西是干啥用的，它都不应该出现在那个房间里。”\
\
“地下人的东西平白无故出现在地上，你觉得那是为什么？”\
\
“这……可就不好说了。就算是想要推理，咱也没有那个脑子。至于赭石呢，还年轻，对什么都有兴趣，看到奇奇怪怪的石头就想捡，没想那么多。谁知道这竟然招来了人类”高附低头叹气。“要是可以问，我还想问呢——人类找地下人的东西是要干啥？”\

“有什么都随便吧，我手头跟人类有关的事儿够多的了，人类以外的东西即使存在也懒得管。我是在调查曾经携带过这个蛋的人类。”\

“人类？听你这意思，是人类把这个东西带到房间里去的啊？”\
“但是那个晚上确实啥也没……啊，等等。在那一夜之前我们倒是目击到了人类进入那个房间，这算不算？而且啊——”\
“唔唔，原来如此。这就说得通了……总之，那个人类遭到袭击，失去了灵魂。”\

“你们见到了——人类？活着的时候？”\
\
“那可是最坏的体验了，想想就让人头疼。不说别的，赛兰今天之所以要俘虏你的朋友，多半也是因为那天吃过亏：放着人类不管，事情肯定会变得棘手。我不是不能理解他”\
“就在赭石捡这些劳什子的几天前——四五天前？和今天一样，我们要去猎鲸，却遇到了在原处转转悠悠的人类。这害得我们没法前进，只能原地等人类走了再说。和今天不同的是那天晚上鲸出现在了人类面前——然后你懂的。”\

副会长脸色发青。确认到这一点，高附试图把故事讲得更加活灵活现。\

锵。\

副会长这才意识到自己的失言。\

毫无防备地倒退两步，副会长差点在大战步枪上摔倒。\

\
“没错，他——要有什么能把他变成少女，那可能确实只有魔法了。但我猜‘魔法少女’应该不是这个意思吧？是吗？”\
“——那天夜里，死在塔楼上的好汉是个男娃子。”\
