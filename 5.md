挂坠是个透明的小盒。挂在兔子胸前时，就像是一块勋章。微光沉淀在碎片底部，已经很不安定，随时都会熄灭。赭石慢慢从脖子上摘下挂坠，在人类注视之下轻轻放在地面，然后后退着返回兔群中间。副会长眼光追随着赭石的一举一动，脸上的表情丝毫没有舒缓。\
\
副会长手无寸铁，却震慑住了以军师为首的一群兔子。她提出要查看赭石的挂坠盒，后者不敢不答应。装着发光碎片的挂坠盒已经放在副会长眼前，她却并不急于取来观看。\
\
在她背后，朋友靠墙坐下来休息。\
\
她没被夺来的兵器的拉胯枪托捶断骨头。但是大战步枪原本就不是为立射设计，端起来开火两次足以令人身心俱损。装填好的第三发子弹原先是立刻转向，枪口直捅没回过神来的游击队长。幸而副会长及时赶到，调停了这场争端。人类和兔子互换了人质，朋友得以被释放，而受到严重惊吓的赛兰被两只兔子搀着返回了队伍后方。\
\
“哇哦，赛兰这家伙怎么这么怂，他腿都软了耶”\
\
“得了。那枪口比脸还大，换了你也得怂”\
\
一老一少两只林地兔交头接耳。赛兰的确哆哆嗦嗦，却还是强打精神。连带着赭石也无暇关注交出去的挂坠盒，而是充满担忧地回望队长的背影。\
\
“拉斯普京呢？”\
朋友不按路数，比起刚刚才和鲸对峙过的副会长，首先关注了不在场的白兔的去向。\
\
“他说他腰断了走不了路，所以我把他丢进清扫间的水桶里了”\
\
朋友的脸色前所未有地严峻。\
“盖严了吗？”\
\
“你尽管放心，他要能跑出来算我输”\
\
“你当时也应该直接回去。子弹虽说只有一匣，也该够你破门了。”\
\
“谁会想到用这玩意儿破门？？”\
副会长不再对对话抱有期望，进而转过眼光，被注视的赭石下意识地一激灵。\
\
“你老实交代，这个东西——这堆碎片，是你从哪儿搞来的？？”\
\
眼下的栗毛垂耳兔的抖得说不成话。圆圆的兔眼里瞬间溢满了泪水。\
\
“你倒是说点什么啊——别哭，站直了说话！”\
被逼问的小兔子原本还能勉强忍住眼泪，副会长的警告却起到了反作用。无可奈何地看了一眼怎么都拒绝说话、现在还扑扑簌簌落泪的栗毛兔，副会长坐回地面，像是自言自语，又像是在向背后的友人抱怨。\
“看来拉斯普京说的是真的，兔子确实没胆子跟人说话”\
\
讯问一筹莫展，副会长从打开的挂坠盒里拿出发光的碎片放在手心。有着能够边缘的两块碎片拼起来能形成一个蛋状的底部，比想象中的还要重——和它相比，透明挂坠盒就像塑料纸一样没有质量；摸起来的质地则像是珐琅，只是微光能从几近透明的表面渗出。\
“好吧好吧。那你告诉我，今晚你们怎么抓到那边那个人的？这总可以吧？”\
\
“咳咳。这个嘛，就让我来解释吧”\
队伍里的高附清了清嗓子挺身而出。\
\
“等、等等，赛兰抓人的时候咱俩不是根本没在场吗！”\
\
“这有什么办法，队长和军师都被吓傻了，现在只有我这个老兵能跟人类交涉了不是么”\
一老一少两只林地兔子边说相声边站到了副会长眼前。\
\
“拜托你交涉的时候别瞎编”\
副会长不抱希望，以一边膝盖为支点的手臂撑住了腮，垂下眼来看地上挺直了腰杆、兔耳有别人两倍长的成年林地兔。\
\
“哎哟，小姑娘，瞧你这话说的。我都没看见，可不就是只能瞎编吗？”\
“但好在今晚的运输计划需要和本队配合一致，所以赛兰他们在哪儿干了什么，我心里都有底”\
在地下边境的生活和从事运输业的经验使得高附即使应对起人类来也并不紧张。\
\
“赛兰呢，纯粹是迫不得已。”\
“时间一到，你们这个学校内部就会化为迷宫。这一点小姑娘你已经体验到了吧？在这迷宫里，底层的雕像后的这道楼梯只向上下延展，是重要的垂直交通线。若是没能在迷宫化开始前登上这道台阶就会迷路。人类是如此，我们也一样。\
“今天的学校里本应该在实施宵禁。原计划的出发时间里，整座学校里都不应该有人，赛兰他们能顺利从大厅到达那个楼梯口的雕像……本该是这样的。没想到，那里却出现了个人类，就是小姑娘你被抓住的朋友啦。”\
“我跟你对天发誓，赛兰虽然脑子缺根筋，但绝对没傻到主动跟人类挑起冲突。要不是你的朋友待在那儿不肯让路，他也不会铤而走险。说实话我直到刚才都纳闷儿他究竟是怎么俘获她的——那些个豆子机炮除了打打地下人，对人类来说应该不算什么威胁才对。不过，”\
目光越过副会长，高附瞄了一眼远处坐在墙下休息的人类。高窗投下的光不够明晰，人类身体的大半陷入阴影之中。被她夺走的大战步枪倚墙而放，退掉的子弹和空了一半的弹夹则放在手边不远处的地板上。\
“——你的朋友，敢情也是将计就计啊。”\
\
“你说这都是为了登上台阶……然后呢，你们登上台阶要去哪儿?”\
\
“这还不明白吗？当然是去猎鲸啦。”\
\
……副会长皱起了眉。眼前的砂色毛球的确比被她安置进楼下水桶里的拉斯普京看起来更加体态健硕，但要上称约，大概肉也多不了几两。在他背后，组成游击队的兔子也不过几十只。相比而言，鲸——即使是刚才只有牛那么大的幼鲸，发起怒来也足够把他们踏得全军覆没。\
\
“……就凭你们……不是，为啥要猎鲸？？”\
\
“啊呀，哪里有那么多为什么，你是‘绿色和平’吗？”高附愁眉苦脸地挠了挠耳朵。“旃陀罗兔一族自古以来就有和平利用鲸的文化，是继承和发扬传统的一部分”\
\
“啥是‘和平利用’？”\
\
“呃，就是……就是收集鲸的尸体。好吧，骗你的。我们那个屯本身已经因为鲸瘟爆发灰飞烟灭了，所以可能也就没什么文化可供继承发扬了。除此之外，赛兰的先祖是地下人所说的‘蛮夷’，早就被驱逐到了地面上，得跟鲸战天斗地才能活下去。就连地下人不也是跟鲸有不共戴天之仇吗？还有你们，猎鲸可也算是在帮你们的忙，算是做好事不留名吧”\
\
“要是你们今晚没劫持人类，这话说不定我就信了呢？”\
\
“哎哟，我不都说了吗，全是误会。再说，好像被坑了一道的是我们吧？赛兰吓得差点背过去，还被你们缴了械”高附耸了耸肩。\
\
副会长从鼻子里哼出一声。\
“那还不是因为你们昏了头，麻烦找到了人类身上。”\
\
“这我倒是……无话可说。说实话，我刚才要是在场，拼了命也会拦住赛兰做这种傻事”\
“……但你看，你们也不该大晚上的到这儿来。既然都不占理，我们应该握手言和才对啊。”\
\
“能不能握手等会儿再说吧。我到这儿来，有正儿八经的原因。要不然，谁会想在这种夜里到处瞎逛”\
“我是在找这些碎片。”\
\
刚才把挂坠盒挂在胸前的小兔子仍然眼泪汪汪，合不拢的三瓣嘴看起来还是没法吐出一个字，还被不自觉地面呈凶相的人类少女吓得一激灵。\
“……不不不，这也还是我来解释吧”注意到副会长表现出的露骨不信感，高附赶紧打圆场。“赭石他不太擅长跟人类说话”\
\
“你倒是挺擅长的啊，嗯？”\
\
“咳，跑运输嘛。要吃这碗饭免不了跟各路人打交道，容不下社恐。”\
“这玩意儿嘛，是你们小娃子所说的‘春季学期’的第二天，在西边那座楼后面的沟渠里捡到的。”\
“……你先别急着反驳我。我也看得出来，这不是沟渠里随处都滚着的普通石头子儿，肯定是从什么地方掉到那里去的。但你要问它一开始是在哪儿，我爱莫能助。那道沟的源头是山顶雨水，何况还是条明渠，从哪儿捎来了那些碎片都不奇怪。”\
\
“春季学期第二天……那天早上学校里发现了人命事故，你听说了吗？”\
\
“你这话说的就好像你们这儿哪天不发生命案似的。……不不不，别那么瞪我。虽然不至于天天有，可你要说是哪一起，那我哪儿能想得出来。”\
“……啊，瞧我蠢的。摆明了只能是那一起——我想起来了。只有那一起命案惊动了其他人类。我听说他们呼朋唤友地去看了那个现场呢——就是西塔楼上发现了个尸体的那次吧？”\
“……小姑娘？你还好吧？”\
\
“你还知道……别的命案？”\
\
高附用一只爪子擦了擦鼻头，思虑沉重地吐出一大口气。\
“唉，小姑娘。这里可是……契丹鲸的栖息地，它们是靠吃人类活着的，你不会是对此一无所知地跑到了这儿来吧？”\
\
“可、可是，那些个尸体——难道是被”\
踌躇不决的副会长脑中闪现出了没有脸的蹄鲸啃食同类尸体的场景。\
\
“在靠近地上的外围，‘贱民’聚集的村落不计其数。”\
“已经消亡的旃陀罗兔是其中之一。这些聚落的居民，除了为了生计收集鲸尸之外，还有责任处理地上的……鲸以外的东西的尸体。”\
“唉，其实我也纳闷儿。说实在的我还是第一次见人类在那个塔上遭到袭击，那儿实在太高了，估计是出来搜集尸体的村民也没能一晚上打个来回，尸体才会留在那儿。”\
\
“你看到了那个人类被鲸袭击的场面？”\
\
“那可是最坏的体验了，想想就让人头疼。不说别的，赛兰今天之所以要俘虏你的朋友，多半也是因为那天吃过亏：放着人类不管，事情肯定会变得棘手。我不是不能理解他”\
“就那个晚上，我们要去追锁定好几天的猎物，却在猎场遇到了在原处转转悠悠的人类。这害得我们没法前进，只能原地等人类走了再说。和今天不同的是那天晚上鲸出现在了人类面前——然后你懂的。”\
\
副会长脸色发青。确认到这一点，高附试图把故事讲得更加活灵活现。\
\
“——遗憾。你要问我什么感想，那就是遗憾”\
“你想啊——都吓坏了，手脚都不听使唤，稀里糊涂就得面对怪物，可能都没搞清楚自己面前的是什么东西。你可再也没法看到比那个更可怜的场面了。”\
“对面的鲸呢，看样子也饿了好几天。看见猎物都眼冒绿光了。人类挣扎时推倒了一地的家具它都没注意到，一蹄子就踏了个粉碎。唉，真是的，该有人教教这些小娃子，半夜里别往学校里闯。”\
\
——脑中不由自主地出现了亲友小千被逼至绝境的景象。\
“然后呢？”\
\
“然后，那头鲸就把小娃子的灵魂吃了呗。还算是电光火石，那小娃子转眼就没气儿了。对了，我跟你说过没？鲸要弄死你们可用不着到眼皮子底下来，只要灵魂出窍，那可就救不活了。”\
“麻烦事儿还没完。这鲸吃饱了人类的灵魂，精神头儿也就上来了，对我们而言就比饿着的时候难抓多了。它要是莽上来，炸药包和燃烧瓶肯定没用——以前我们试着这么做的时候可是误炸了不少自家弟兄。我们只好无功而返，计划等人类的尸体被回收了以后再回来设陷阱捕鲸。这可比追击鲸不靠谱多了——但好在还不是一无所获。”\
“所以你看，人类不该上这儿来。即使这些碎片如你所说是被那个牺牲者带到了房间中的，它们也是地下人——也就是，我们的同族——的东西，人类拿着也没有好事发生。我不知道你找它干啥，但假如小姑娘你够明智，那就应该——”\
\
锵。\
\
安全阀从接触回路分离的声音从背后传来，令副会长和兔子们短暂分神。坐在墙下的朋友从回路接口上拔出空弹匣，然后同样大力插上从裙子口袋里掏出来的一模一样的十发装新弹匣。回路重新连接，兔群不约而同地下意识捂了捂耳朵。\
\
“你再跟我详细说说那晚的人类”副会长重新看向地上的林地兔。“……她没试着逃走吗？”\
\
“没。唔，怎么说呢”高附摸了摸下巴。“虽然我相信那更可能是被吓得走不动道儿了，不过……”\
“那个小娃子，还真算得上是英雄豪杰——拖住了那头鲸，让朋友有机会逃走，为此牺牲了自己”\
\
“另一个人？？”\
\
“啊，对了。一开始是有两个小娃子来着。不过另一个人逃得挺顺利的，所以……”\
\
“怎么可能有另一个人！”\
\
“啊呀别那么大声。你倒是说说怎么不可能？我对天发誓，我说的句句属实。要有一个字瞎话，那就让我得鲸瘟，然后天打五雷轰”\
\
那并不是没有可能，副会长深知这一点。而且这么一来，反倒一切都有了解释：和拉斯普京的证词相悖，小千几乎不可能是成功的魔法少女。更何况她毫无经验，在初次见到的怪物面前，只能束手就擒。\
\
——但她肯定会像这样保护他人，这一点毫无疑问。她就是这样一厢情愿、令人想起来就心乱如麻。\
\
“行了。”从地上站起身，感到眼前有些眩晕。“你说的我明白了。剩下的就让我找到那个溜掉的人，从他嘴里审出来吧——”\
\
“虽然不知道你为什么对这事儿有这么大的兴趣，不过你想找到那个小娃子可不容易。你也知道吧？这学校里的人类可比鸫河滩上的沙子还多”\
\
“那可不劳你费心。学生会这点本事还是有的”\
副会长抚平起皱的裙子。转过身来，看到她的朋友并不急于起身，又心生不满。\
“……不过，无论这人是谁，他真够能沉得住气的。他可是先目击了蹄鲸又被魔法少女救了一命，这么久过去了学校里竟然没有一点传闻……你呢？第一学群里有没有什么风言风语？”\
\
她的朋友如预想中一样地摇了摇头。地上的林地兔毫无征兆地爆发出嗤笑。\
\
“‘魔法少女’？？？什么鬼？？？”\
\
副会长这才意识到自己的失言。\
“……听着，不是我想这么叫她的——”\
\
可是高附乐不可支，根本没留意副会长的分辩。他抬起右爪来抿掉笑出来的眼泪，才意识到对面的人类已经有些生气了。\
“现在魔法少女门槛已经这么低了？这难道是什么最糟的平权运动？？啧啧，人类世界可真是文恬武嬉”\
“——你可能是搞错了什么。当然，也可能是你们太过进步，我理解不了”\
“不过我还是要提醒你，我跟你说的那个人多半当不成魔法少女。不，要是那个小伙子是背负着魔法少女的身份去死的，我倒有点怜悯他了。”\
\
“你说‘他’——”\
毫无防备地倒退两步，副会长差点在大战步枪上摔倒。\
\
\
“没错，他——要有什么能把他变成少女，那可能确实只有魔法了。但我猜‘魔法少女’应该不是这个意思吧？是吗？”\
“——那天夜里，死在塔楼上的好汉是个男娃子。”\
