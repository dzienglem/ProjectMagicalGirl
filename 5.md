挂坠是个透明的小盒。挂在兔子胸前时，就像是一块勋章。里面装着的碎片看似原本属于一个球体。微光沉淀在碎片底部，已经很不安定，随时都会熄灭。赭石慢慢从脖子上摘下挂坠，在人类注视之下轻轻放在地面，然后后退着返回兔群中间。副会长眼光追随着赭石的一举一动，脸上的表情丝毫没有舒缓。\
\
副会长手无寸铁，却震慑住了以军师为首的一群兔子。她提出要查看赭石的挂坠盒，后者不敢不答应。装着发光碎片的挂坠盒已经放在副会长眼前，她却并不急于取来观看。\
\
在她背后，朋友靠墙坐下来休息。\
\
她没被夺来的兵器的拉胯枪托捶断骨头。但是大战步枪原本就不是为立射设计，端起来开火两次足以令人身心俱损。装填好的第三发子弹原先是立刻转向，枪口直捅没回过神来的游击队长。幸而副会长及时赶到，调停了这场争端。人类和兔子互换了人质，朋友得以被释放，而受到严重惊吓的赛兰被两只兔子搀着返回了队伍后方。\
\
“哇哦，赛兰这家伙怎么这么怂，他腿都软了耶”\
\
“得了。那枪口比脸还大，换了你也得怂”\
\
一老一少两只林地兔交头接耳。赛兰的确哆哆嗦嗦，却还是强打精神。连带着赭石也无暇关注交出去的挂坠盒，而是充满担忧地回望队长的背影。\
\
“拉斯普京呢？”\
\
“他说他腰断了走不了路，所以我把他丢进清扫间的水桶里了”\
\
朋友的脸色前所未有地严峻。\
“盖严了吗？”\
\
……可惜副会长并不想搭茬。在她展开追击战之前，拉斯普京反复宣称自己腰痛难耐，令她忍无可忍，于是她按照自己心目中的最高规格把白兔厚葬，在水桶盖上还倒扣了放抹布的篓子，大可不必担心他被魔兽找到。但这段说明实在不适于现在进行。无视朋友，副会长低下头，被注视的赭石下意识地一激灵。\
\
“你老实交代，这个东西——这个碎片，你从哪儿拿来的？？”\
\
眼下的栗毛垂耳兔的抖得说不成话。圆圆的兔眼里瞬间溢满了泪水。\
\
“你倒是说点什么啊——别哭，站直了说话！”\
被逼问的小兔子原本还能勉强忍住眼泪，副会长的警告却起到了反作用。无可奈何地看了一眼怎么都拒绝说话、现在还扑扑簌簌落泪的栗毛兔，副会长坐回地面，像是自言自语，又像是在向背后的友人抱怨。\
“看来拉斯普京说的是真的，兔子确实没胆子跟人说话”\
\
“……就你那么凶，光是旁听我都快哭了。”\
\
“什么鬼，我怎么记得以前审你的时候，你可是冷静得不行……呃。”\
讯问一筹莫展，副会长从打开的挂坠盒里拿出发光的碎片放在手心。碎片比想象中的还要重——和它相比，透明挂坠盒就像塑料纸一样没有质量——摸起来的质地则像是珐琅，弧形的表面显然是构成了一颗“蛋”的底面。\
“好吧好吧。那你告诉我，今晚你们怎么抓到那个人的？这总可以吧？”\
\
“咳咳。这个嘛，就让我来解释吧”\
队伍里的高附清了清嗓子挺身而出。\
\
“等、等等，赛兰抓人的时候咱俩不是根本没在场吗！”\
\
“这有什么办法，队长和军师都被吓傻了，现在只有我这个老兵能跟人类交涉了不是么”\
一老一少两只林地兔子边说相声边站到了副会长眼前。\
\
“拜托你交涉的时候别瞎编”\
副会长不抱希望，以一边膝盖为支点的手臂撑住了腮，垂下眼来看地上挺直了腰杆、兔耳有别人两倍长的成年林地兔。\
\
“哎哟，小姑娘，瞧你这话说的。我都没看见，可不就是只能瞎编吗？”\
“但好在今晚的运输计划需要和本队配合一致，所以赛兰他们在哪儿干了什么，我心里都有底”\
在地下边境的生活和从事运输业的经验使得高附即使应对起人类来也并不紧张。\
\
“赛兰呢，纯粹是迫不得已。”\
“时间一到，你们这个学校内部就会化为迷宫。这一点小姑娘你已经体验到了吧？说是迷宫，其实也不是完全没有头绪。比如这道楼梯只向上下延展，还能作为夜间交通线使用。不过，若是没能在迷宫化开始前登上这道台阶就会迷路。人类是如此，我们也一样。”\
“今天的学校里本应该在实施宵禁。原计划的出发时间里，整座学校里都不应该有人，赛兰他们能顺利从大厅到达那个楼梯口的老头儿雕像……本该是这样的。没想到，那里却出现了个人类，就是小姑娘你被抓住的朋友啦。”\
“我跟你对天发誓，赛兰虽然脑子缺根筋，但绝对没傻到主动跟人类挑起冲突。要不是你的朋友待在那儿不肯让路，他也不会铤而走险。说实话我直到刚才都纳闷儿他究竟是怎么俘获她的——那些个豆子机炮除了打打地下人，对人类来说应该不算什么威胁才对。不过，”\
目光越过副会长，高附瞄了一眼远处坐在墙下休息的人类。高窗投下的光不够明晰，人类身体的大半陷入阴影之中。被她夺走的大战步枪倚墙而放，退掉的子弹和空了一半的弹夹则放在手边不远处的地板上。\
“现在我倒明白了。你的朋友敢情也是将计就计啊。”\
\
“你说这都是为了登上台阶……然后呢，你们登上台阶要去哪儿?”\
\
“这还不明白吗？当然是去猎鲸啦。”\
\
……副会长皱起了眉。眼前的毛球的确比被她安置进楼下水桶里的拉斯普京更加健硕一些，但要上称约，大概肉也多不了几两。在他背后，组成游击队的兔子也不过几十只。相比而言，鲸——即使是刚才只有牛那么大的幼鲸，发起怒来也足够把他们踏得全军覆没。\
\
“……就凭你们……不是，为啥要猎鲸？？”\
\
“啊呀。哪里有那么多为什么，你是‘绿色和平’吗？”\
高附挠了挠耳朵。“你就说我吧。林地兔一族自古以来就有和平利用鲸的文化……虽说这个族本身已经灰飞烟灭了，所以可能也就没什么文化了……赛兰的先祖是游骑兵，地下人管他们叫‘蛮夷’驱逐到地面上。如果不猎鲸，他们连生存空间都受到威胁。就连地下人不也是跟鲸有不共戴天之仇吗？”\
\
“‘和平利用’是啥？”\
\
“呃，就是……就是利用鲸的尸体。用鲸鬃做刷子啥的。”\
“当然，那时候让我们村的老少爷们儿去猎活着的鲸，是太难为他们了。但现在可不一样。游击队的火力足够对鲸造成伤害，刚才的大战步枪你也看见了吧？……虽然还是被它跑了。不过，以前的燃烧弹可是打出过实打实的战果。”\
\
“你们在学校里，用燃烧弹？？？？你吃豹子胆了？？你——”\
“等等，你说的战果，该不会就是——”\
副会长抓起发光的碎片。\
\
“没错，就是找到这块东西的地方。那个尸体还没来得及回收，本来是打算今晚趁机去一趟的”\
高附在地板上点了点他那对大脚。\
“话说你还真行。调查得还挺彻底，我还以为人类铁定会被那个房间吓疯呢。”\
\
“别说傻话。我今晚可都已经对付过两只鲸了。”\
\
“倒不是那个可怕，而是……”\
“啊……我忘了。你们人类听不到‘鲸歌’啊。”\
\
“鲸歌？”\
\
“啧啧。小姑娘，听我一句劝。虽然你的勇气值得嘉奖，但有些事情还是别打破砂锅问到底的好。就像我说的那样，搞不好就会被吓疯呢。”\
“今晚你和你的朋友也许的确击退了鲸。但是，那就像是一无所知地安然穿越雷区一样，纯粹是天公疼憨人。一旦知道了脚下埋满了地雷——你可想想吧。那时候你自己可都能把自己吓死。”\
\
锵。\
\
安全阀从接触回路分离的声音从背后传来，令副会长和兔子们短暂分神。坐在墙下的朋友从回路接口上拔出空弹匣，然后同样大力插上从裙子口袋里掏出来的一模一样的十发装新弹匣。回路重新连接，兔群不约而同地下意识捂了捂耳朵。\
\
“哇，有，有话好好说嘛。”高附也意识到了一直不发一言的人类的态度。“好吧好吧。鲸歌就是契丹鲸的叫声啦。哎哟，想我年轻的时候也纳闷儿，怎么鲸都快把喉咙喊破了，人类的小娃子还就愣是杵在那儿等着被杀掉呢？敢情是他们压根没听见。”\
