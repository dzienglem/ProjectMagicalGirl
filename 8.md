接触射击制度意在校内恢复被精英病打乱的教学秩序。发作期间患者的肉体发生变异而又完全丧失理智，攻击性极强的同时对生物镇静剂产生抵抗。乙型阻断剂是少数能对发作期患者产生效用的药剂之一，却会对受体精神造成不可逆的损伤。在集中制义务教育的顶点鸫丘中，这种精英病后遗症就意味着从金字塔上永远跌落。\
\
这个后果使得阻断剂在问世当初就被学生反对。秩序固然宝贵，但无人希望成为宝贵秩序的轻贱代价。在和学生代表的胶着论战的最后，“三发规则”作为学生意志的集中体现得以固定。人们希望借由这种对阻断剂执行人的限制，来保障学生权益。另一方面，规则换来了阻断剂在校内的通用，节约校医资源的同时最大限度维持了学校机能。规则同时降临在天平两端，使得人们心安理得地享受到了平静，以至于双方都简直是刻意地忽略了，规则中其实存在着的漏洞。

“啊？……这可就奇了。你的意思是，当初你就知道这些碎片能拼成的东西是什么，而你又见过它，所以才把它们捡回来？”\

“我，倒是不认识她们说的什么‘石榴挂坠’……”\
赭石慢慢低下头。\
“是这些碎片不一般。”
***
\
昔日人类为掩埋作业而设置的临时配电室早已完成使命，优越的立地条件却使它在废弃后也依然大放光彩。这座依仗地势的钢铁建筑能够绰绰有余地俯瞰洞底水潭，以及环绕坑洞的整整一壁面的栈道。房顶铺密管道，只在边缘留下空地。放下细长的大战步枪后，人类要想在那里坐下还必须把双腿搭在房檐之下。迎面是穿过整个洞穴的地下热风裹挟着沙尘，也被人类拽下来的安全生产标语挡住。\
\
赛兰和鬲昆聚精会神，四只眼睛直盯着人类手里的门把手。由门把手控制的双向鱼线收紧，步枪尾部的螺栓随即被带动旋转、沿着直线轨道向后滑动。失去压迫的塑料书夹弹簧在膛室正上方松开。接下来，一侧鱼线被伸至极限而固定，反向的另一侧把螺栓拖回原位，直至把手沿着刻好的沟槽滑落，螺栓完全闭合。放开滑轮，人类扣下扳机，看不见的膛室中击发槌弹出的声音清晰可闻。螺栓砰地向后弹出，重新滑入沟槽。两只兔子同时瞪大了双眼。\
“哦哦哦，厉害！”\
\
“哦哦哦，厉害！等等，这不就成了个鱼竿了吗！”\
赛兰挠了挠头。但接着就立刻爬到人类膝上，仔仔细细端详被改装一新的大战步枪。\
“我看看。就这么转这条线，就能让它拉紧，大拴就能自己掉出来，然后……”\
\
“现在这个只是小样”\
人类轻声细语，同时默许了小兔子站在自己怀里。把步枪枪口架在身旁的锅炉管道上，她转身把手伸进散放在一边的校服上衣。从那里摸出来的接触枪已经解除了整个缓冲罩，只剩下裸露的、把中心回路缠得密密实实的限制元件。\
“实际做的时候，没有这根线，也不会有这堆夹子。但是得把肩托锯短，给这个接触回路腾出地方。”\
\
“可是，可是！”\
鬲昆还不习惯和人类说话，因此非常激动。“明明下面那个坑里，不就有现成的‘街杵回路’吗！”\
\
“对，我记得……你说过，你是上面小娃子们里的‘射手’什么劳什子对吧？要是你把这把枪里的接触回路拆掉了，那学校那边……”\
赛兰犹犹豫豫，似乎想要试探人类。可对方并不为所动。\
\
“掘进机接触回路的标定功率过于大了。……就是说，它没法装在这杆步枪上。”\
“而这把接触枪现在的状态，也肯定没法继续在学校里使用。所以不要紧”\
\
“那要是那样！你干嘛要让老高他们，掘下面的那个，‘街杵回路’呢！”\
\
“哇你这个小鬼，可长点儿记性吧。要说几遍你才能明白啊？”\
赛兰不以为然，从人类的膝头跳下，对鬲昆劈头盖脸。可惜后者尽管年幼，却不吃这一套——\
\
“那还不是因为赛兰，解释得一点不利索”\
\
“嘿！说了几次了，别对大人直呼其名。听好了，我再给你讲最后一次”\
\
“才没人想听赛兰讲的没头没脑的话呢！”\
\
赛兰气急败坏，想要捉住趴在一边的鬲昆。鬲昆仗着身量小而灵活躲过，害得赛兰扑了个空面部着地。鬲昆得意忘形，差点从房檐上掉下去，幸而人类早有准备，把他接在手中。\
\
“话说，你找到那个人类了？”\
鬲昆被放在了赛兰身边，他抬头向救了他的人类搭话。\
\
“那个人类？”\
\
“就是那天，在那个塔上被鲸吃了的小娃子”\
\
人类直起了身。\
“他下落不明这件事还没被发觉，但学校做出逸脱判定应该就在几天之内。我已经查到了身份和姓名”\
\
“哇哦，可是尸体不还没找着吗？你怎么知道他是谁啊？”\
\
“倒也不是什么难事儿。他是第一学群的人”\
\
“你见过他吗？”\
\
“她也见过”\
\
塞兰坐起来盯着人类，一手扶着下巴。\
“这么说他是你说的，那个什么茶会的客人了？”\
\
“不然他也不会到那座塔上去。”\
\
“小娃子被鲸杀死，是我亲眼看到的。所有人也都看到了。我还看到了女娃子在他背后逃出了房间——可却是女娃子的尸体在房间里被找到。小娃子蒸发了，居然！”\
\
“尸体没有可能被鲸吃掉吗？”\
\
赛兰一脸苦恼。\
“那可没法保证。按说，它们并不爱吃人灵魂以外的东西——那种家伙虽然个子大，牙口其实一点不好，要说它能一晚上就把一个人吃得干干净净，那我可不信。”\
“可我们还没找到那头鲸，谁知道他肚子里有没有躺着个小娃子，万一呢”\
\
“你说，你说……那堆发光的石头是那个逃走的女娃子的东西？”\
鬲昆转了转眼珠。\
\
“嗯。把它交给她的白兔说，挂坠能给她打倒鲸的力量。”\
\
“这不扯淡呢嘛，地下人哪有这本事”\
塞兰翻了个白眼，一边不自然地挪了挪屁股。\
\
“哎哎，那么一来，她发现上了地下人的当，所以那时候才逃走？这不就说的通了吗！”\
“还把石头丢在后面的沟里，才被赭石捡到”\
\
“要是那样，她就不会再回到房间里还死在那儿了。”\
\
唔姆。两只兔子同时低下头陷入不着边际的沉思。片刻后还是塞兰先开了口——\
“你听了那地下人的话潜入塔里，是为了调查那女娃子的死因吧？”\
\
人类不紧不慢地从步枪上拆掉鱼线。\
“当时我并不怀疑他讲的关于鲸的事。她死前并没有预兆，死后也没有留下痕迹。如果有什么袭击能让人失去灵魂而死，那能解释她的死状。”\
\
“那就是为了找那个‘石榴挂坠’？我记得，呃，她……是这么说的？”\
\
“我想调查沙龙。”\
\
鬲昆满脸写着困惑，赛兰却恍然大悟。\
“——哦哦哦，怪不得。那里是鲸的巢穴，但你们人类不知道。所以才会纳闷儿那里净发生怪事”\
\
“今年以来，那个沙龙的参加者里已经有三个人丧命。那里肯定发生着什么”\
\
“可你自己不也在那个茶会上么？要是有什么蹊跷，你能注意不到？”\
\
“那里倒是……什么也没发生。”\
“但肯定发生了我不知道的事情。只说四月一日那一晚，他们会在夜里出现在房间里，本身就在我意料之外。”\
\
“就是说，你想调查在那里发生过，而你又不知道的事儿？”\
\
“可我没能找到什么。那里没有任何被藏匿在房间里的东西。要是那样的话……”\
“那里可能有什么只存在于一部分成员之间的秘密盟约。参与者自然会把盟约对**她们**保密，我也就没法知道。甚至于……”\
\
“甚至于，啥哦？”\
始终昂着头盯着人类的鬲昆终于眨了眨眼。人类却止住话头，低下头默默地拆下固定鱼线用的一排书夹。\
\
“……你，你这崽子，怎么话只说一半，是想急死谁啊？”\
\
“她在那个夜晚留在塔楼里是反常的。但是，现在想来，那也许是因为她那时已经开始注意到，沙龙中有问题。”
***
\
校医院占据了主塔楼的整个六层。磨砂玻璃幕墙隔出入口，里面就是鸫丘中最大的教学外学校机构。幕墙的对面人影幢幢，而休息室却空无一人——校医院中不是病患就是工作人员，而鲜有“访客”。副会长挂掉自助式通话器走到窗前。面向主楼前广场的双层玻璃窗外，雪下个不住，以至于积压的雪的厚度已经超过了窗框本身。她书包里的拉斯普京发现四处无人，探出头来。\
\
“啧啧、竟然直奔医院就来了。也难怪，那孩子可是长得剑眉星目，俊得很”\
\
“得了吧，谁还没长个眉毛眼睛”\
副会长对拉斯普京的浅薄嗤之以鼻。\
\
“那不一样。人家不光长得一表人才，还知书达理，文质彬彬。‘维罗纳的夏天找不到这样一朵好花儿’。”原先就慈眉善目的拉斯普京此时更是眉开眼笑。\
“刚才你也看到了，身手还不错呢。不愧是个‘射’——”\
\
“再啰嗦我就把这窗户打开，把你从这儿丢下去”\
\
拉斯普京一边讪笑，一边缩回书包里。正巧此时背后的幕墙向两边分开，校医院的接待员出现在自动门后。
***
\
“实在意外，学生会长竟然亲自跑一趟，就为了还我这个。”\
\
“别客气。反正我也是，顺便”\
\
“……顺便，来校医院？”\
\
拖着绷带的射手满脸都写着疑惑。小观察室位于平层中部，是从诊室用石膏隔板隔断的简单小屋，顾名思义，是诊疗全部结束前供伤员静息观察的地方。被刺伤皮肉的射手无需复查，只是在此处等待护士为他带回干净衣物，并趁机接待来访的副会长。\
\
感觉到白兔无声地笑得浑身发抖，副会长默默用胳膊肘捅了捅自己的书包。\
\
“算，算是吧。你有大碍没？校医怎么说？”\
\
“我没事。校医来得很迅速，及时止了血。缝合手术也很简单。”\
“他们说我可以去参加下午补上的文学课了。”\
\
“要是那样，你可以继续歇着了。刚才的文学课预定周五再补，下午的文学课是七班。”\
\
观察室里只有一把椅子，是供伤员休息用的，于是访客只能站在椅子对面的矮屏风旁边。来自高处的视线能轻易观察到伤员的两肩陷了下去，副会长不以为意。\
\
“——他们也停课了嘛。这也是当然的事情。”\
\
“是啊。”他与其说是应答，不如说是在喃喃自语。\
“今天的课上其实也很危险。没有扩大伤害都只是运气。”\
“但是，下一次……，我不知道我还能否应对。”\
\
“校医院也是时候拿出个应急方案了。这样当然没法对付将来的集群发作。”\
\
“将来的——？”\
射手大吃一惊，忍不住抬起头来。他注视下的副会长在屏风后抱起了双臂。\
\
“集群发作在各个学群都变得日渐频繁，对吧？担心这个的，可不止是校医院。”\
\
“……我还以为这仅仅是射手和校医院间的见解”\
\
“啊，这事儿靠一个校医院可搞不定。学生会作为学生代表，自然要共享情报。”——虽然她上次过来通气已经是两个月前调查第一学群人命事件的时候，并没有共享什么有用的消息，甚至也不是以学生会名义——把这些实情隐瞒起来，只说了冠冕堂皇的部分，副会长清了清嗓子。\
\
“那么，学生会的意见是什么？”\
\
剑眉星目神情诚挚，打了个措手不及。副会长有点儿慌。\
\
“学生会？……唔，学生会——”\
“希望尊，尊重现场的看法咯。射手才是最接近精英病的第一线嘛。”\
\
对方低头陷入沉思。副会长暗暗松了口气。\
\
“……恐怕现场也提不出什么主张。校医院似乎曾经提议把学校活动拆分成更小规模——更小的课堂，更细分的生活设施，试图以此收敛每一起事件的影响规模。”\
\
“这听起来不还有点儿道理吗，还能顺便预防传染，简直是一举两得”\
\
“假如精英病真传染的话，那确实”\
伤员像是自暴自弃一般吐出暴论。\
\
“它要不传染，哪里来的集群发作？”\
\
“可它通过什么传染？”射手没受伤的一边手臂垂在椅背。“空气，飞沫，皮肤接触？”\
“防疫手册上虽然写着有传染可能，可校医院从没找出过什么精英病菌。\
\
“它是——精神疾病，自然不会握个手就传染，这人们都懂。但，一撮人同时开始发病的例子出现的次数都快数不清了。不管它是通过什么，这怎么看都是传染嘛。”\
\
射手整个身体深深陷入扶手椅，看起来比控制文学课现场时还要疲惫。\
“同时发病——‘集群发作’现象，未必是在患者间发生了相互传染。”\
“我和您说，已经有越来越多的*射手*已经意识到了这点：后发作的患者，很可能并没有受到率先发作者的影响。”\
“毕竟，正像您意识到的那样，精英病是精神疾病。这一点毋庸置疑”\
\
“这可就奇了。要是那样，集群发作的原因是什么？你们怎么认为？”\
\
射手缄口不言，注视着天顶的双眼只是保持着空虚。\
\
“校医院没有认可那个猜想？”\
\
“我想，是没法认可。尽管在道理上，这应该显而易见，尤其是对每天都要收治病例的校医院来说。”\
“总之，我还没有听说医院针对传播途径展开过研究调查。但是，肯定存在一个某种形式的‘诱因’。”\
\
“是令人患病的诱因？”\
\
“不。诱因的效果很可能非常有限，只能在当前场景中诱导发作。但是能被诱导的人，只会是潜在的‘患者’。”\
\
“如果导致发作的仅仅是诱因，那么人们是从哪里得到了这个病？”\
\
“那怕是只有天知道。最坏的情况里甚至包含先天因素，我们对它了解甚少，更无从防范。”\
\
“……所以，照你那么说，在实际发作之前，人们就已经得上了精英病，早就是‘患者’了。”\
\
射手反过身来，面色凝重地点了点头。\
“唯一的救赎是精英病只在鸫丘中内发作。就算是患者，只要不表现症状，那么一旦毕业，就能避免发作和后遗症。”\
“我想校医院拒绝正视这种可能，是因为有安定校内情绪的责任。但，若是学生会……”\
他像是从冥想中猛地回过神。“——护士去得太久了，这样下去会耽误上课。您跟我直接去护士站领东西吧，麻烦您。”\
\
\
射手起身得干脆利落，哪怕动作似乎还是牵扯到了伤口，令他皱了皱眉。副会长跟在他身后出了观察小屋。外面是来时已经走过一趟的清静窄廊，石膏隔板隔起来的观察室都是一模一样的单人观察室。离开观察区，就是由接待柜台分割，入口和病房区分列一侧的共通走廊。和观察区不同，共同走廊是前往所有区域的必经之路，交通量非常可观。\
\
副会长来时并没有经过从接待柜台到病房区一截的走廊。从这里经过的除了护士，还有相当多的学生出入。来到护士站，值班的护士先是对疏忽道了歉——然后为伤员取出了清理过的校服、装着诊断说明和复诊日程的文件夹，以及大概是装着伤药的特制药盒。\
