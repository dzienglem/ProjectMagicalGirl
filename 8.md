接触射击制度意在校内恢复被精英病打乱的教学秩序。发作期间患者的肉体发生变异而又完全丧失理智，攻击性极强的同时对生物镇静剂产生抵抗。乙型阻断剂是少数能对发作期患者产生效用的药剂之一，却会对受体精神造成不可逆的损伤。在集中制义务教育的顶点鸫丘中，这种精英病后遗症就意味着从金字塔上永远跌落。\
\
这个后果使得阻断剂在问世当初就被学生反对。秩序固然宝贵，但无人希望成为宝贵秩序的轻贱代价。在和学生代表的胶着论战的最后，“三发规则”作为学生意志的集中体现得以固定。人们希望借由这种对阻断剂执行人的限制，来保障学生权益。另一方面，规则换来了阻断剂在校内的通用，节约校医资源的同时最大限度维持了学校机能。规则同时降临在天平两端，使得人们心安理得地享受到了平静，以至于双方都简直是刻意地忽略了，规则中其实存在着的漏洞。

\
在人们不抱希望的时候，春雪最终还是下起来了。\
\
对副会长而言这时机并不很妙，因为她正在上的这堂早上的通识课上出现了群集发病。射手没能成功同时压制两名发作者，课堂被指示解散。校医院应召而来，中弹失能的患者被用担架抬走送往校医院，留下的患者在其他学生和教员撤退一空的教室里血流如注地独自发狂，形如丧尸。在他陷入过呼吸的空当，校医团队七手八脚，手持各自的镇暴钢叉一拥而上。患者被捉拿归案，在教室外待命的第二队校医抹了把汗，把已经捏在手里的室内用催泪弹放回随身小柜。教室隔壁楼梯拐角处，护士在搀扶着受伤的射手慢慢坐下——在他试图射击时，患者突然使用讲台上的花瓶反击。一番抵抗后飞出去的接触枪击碎了窗户。玻璃在两人至近距离爆裂，杂夹着雪花的冷风灌了一屋子——撤退指令就是在那之后发出的。\
\
无论如何，刮着穿堂风的教室没法在这种雪天使用，通识课只得提前结束。一时集结在走廊中的学生逐渐散去，最终只剩下副会长一人在天井落地窗的另一侧瞭望驻留在教室门口的校医。\

“……可怜，可怜。他太冲动了。那个孩子既然已经失去了理智，就绝对不该近身。别说是有个瓷瓶在手边，就是什么也没有，估计也能从他身上咬下块肉来”从书包里探出身来的拉斯普京评头论足。\

“你说的没有选择是什么？”\
\
“他就剩一发阻断剂了，而又没有工夫给他等待时机。他只能接近那个人来射击”\

“唉。”白兔轻轻摇头。“阿弥陀佛，你们这个只给人家发三颗子弹的规则实在鸡肋。射击的人和被射击的人，有谁因此得到幸福了吗？最后不还是都要进医院？”\

“你说得对。我也这么想，而且还有……”\
——三发规则一定是自己这辈子见过的最蠢的东西了。既无视效率，又缺乏目的；这堂课被中途打断，自己不得不立身于雪风阵阵的天井，追根究底起来都是这个规则的错：正如她听说的那样，群集性发作案件已经变得越发普遍，而对此三发子弹的作用只是杯水车薪。\
\
不仅如此，铤而走险还会令射手陷入危难。\
\
仿佛早有预料，医疗队的准备万无一失。甚至都不必立刻赶送医院，在现场就能对伤员实施应急处理来防止伤口感染。副会长从远处盯着射手脱下受伤一侧已经被双方的血里外浸透的校服上衣来接受消毒和包扎，情绪较之刚才已经大为冷静——是与这样的出血不相称的冷静，就像是习以为常。副会长猜想他一定是经常这样受伤，又或者是——\
“……他们全部。”\
\
“你说啥？”\
白兔没听懂她的低声自语，抬头一看才发现副会长一脸阴沉可怖，目光已经不知何时投射进廊柱外的雪幕。这不寻常的春雪下得遮天蔽日，对面的第一学群塔楼几乎消隐不见。\
\
“我看得清清楚楚。射手被刺伤后也没有停止接近那个人，校医院的说明书上可不是这么写的”\
“别看他现在看上去好像不疼不痒。那种时候，两边的精神状态都是半斤八两。”\
\
“啧啧啧。你看嘛，留着这规则有什么用呢？光是跟你一起上学的这几天，因为局面失控而停课的事儿都发生几回了？”\
\
“现在是爆发期嘛。那个人被召回校医院不就是因为这个？”\
\
“所以啊，人都不够用了，规则却不能停，这样对大家可都不好。”\
\
“你懂什么。”\
刚刚被防暴叉控制住的发作者被放在轮椅上抬出了教室。校医把轮椅团团围住，急急忙忙地前去搭乘走廊另一侧尽头的升降机，跟在后面的是护士和被简单处理过伤口的射手。旁观了救护过程的副会长走出天井，朝教室门前进发。\
“要是没有三发规则，那也就没什么射手了。校医是没得办法，才这样作茧自缚。”\
“精英病患者都成了个攻击模式的僵尸了，你自然不希望被僵尸袭击，这个意义上你恨不得患者一发作，射手就当场表演一个万箭穿心。但是，每个人，都其实有可能发作。轮到自己当僵尸的时候可就不希望因为注入太多阻断剂而得上后遗症了。”\
“就算自己侥幸没有发作……射手要是有大量持弹可供挥霍，放任他们在校园里四处溜达也会成为潜在的威胁。这种事情，普通人当然不会答应。”\
\
“照你这么说，这规矩还就改不得了？就算是‘魔法少女’出面也不行？”\
\
“别一口一个魔法少女的，烦不烦”\
书包里面的兔子吃了一记不轻不重的腹拳。自觉无趣，为了掩饰尴尬，副会长伸手挽了挽垂在颈侧的发束。\
“……我不知道。”\
\
“喔唷。那还真……真是意外。我还以为你们肯定谈过这件事儿呢。”\
\
“谈过是不假。但是，”\
“我没法料到她对三发规则本身怎么想。”\
\
“哇，这更意外了。不是，”白兔害怕再挨一个肘击，连忙辩解。“我是说啊，你和魔法少女的交情这么铁，你可以说是了解她的一切，却居然……？”\
\
“谁告诉你我了解一切啊？别净自己瞎脑补”\
“……反正小千她，对精英病和校医院这些事都没插过手，也没什么兴趣。……至少直到发生人命事件之前。虽然并不是对校医院里发生的事情完全没有耳闻……”\
“但她并没有说过想要对规则动手。不过啊，”\
\
“嗯？不过什么？”\
\
“也许她试图做什么却又没告诉我呢？这样我不就没法知道了嘛。”\
\
副会长倏地停下脚步。拉斯普京张口结舌。\
\
“你这……还在对她庇护那孩子的事儿念念不忘啊？”\
副会长和朋友据理力争时，拉斯普京被按死在了口袋中，谈话的内容是精选后由副会长转达的。他已经了解到‘魔法少女’暗中出手，特赦罪人并解散调查组一事。\
“她不也是怕你上火吗？你要是当时知道调查中止的前因后果，准得大闹……呃”\
\
“不是那个”\
\
“那是哪个啊？”\
\
“……你怎么这么啰嗦，我哪儿知道是哪个。”\
\
“好吧好吧。但你看啊，她既然有权力干涉调查，为什么不能修改规则呢？她是——唉，我忘了怎么说。兴许八成背不住就能给你们加几发子弹呢……！”\
\
“这根本不是加几发子弹的问题”\
再三确认校医已经全部离开，副会长重新迈进教室。破窗中吹来的风夹杂着雪花，在另一边也能感到寒气。\
“行，就算是三发再加三发。群集发作现在还只是冒了点儿烟，没火苗子。今天是俩人成一波，下次说不定是六个人？七个人？十个人？那时候怎么办？再加更多子弹？那这个规则不就是在闹着玩儿？”\
\
人们相信三发规则足够保护自己。——正因为它的存在，即使有精英病，校内也安全无事，一如既往。\
改变三发规则的尝试，最终都会动摇这个信念的根基。\
\
“……所以呢，你动它一个试试，你看人们不把这里房顶掀了”\
\
“按你说的，三发规则就是在你们这儿使用阻断剂的基石，”\
\
卸下来的书包被放在课桌上，里面的拉斯普京终于平稳着陆，可以慢条斯理地说话了。副会长在讲台侧的墙边跪下，弯下身来打开高度和腰平齐的通风百叶窗。\
\
“……但是，你不也在怀疑这个规则已经被人打破了吗？”\
“要不然，你也不会上这儿来。对不对？”\
\
“谁说的，我这是，”\
“——助人为乐嘛。有人把东西忘在这儿，我只好帮他们找回来啊。”\
\
把手伸进通风窗，副会长一下就摸到了陷在积雪里的接触枪。射手被刺中时仍试图继续射击，被发狂的患者钳制，手中的接触枪则是被投了出去。副会长避难前特意巡视讲台前，认定枪是掉进了叶片脱落了不少的通风窗。顺利找到目标，她直接坐在地上，比谁都利索地拆掉了弹匣——末端的凹槽里只卡着一枚有着涂装成鲜明橘色的圆柱型阻断弹丸。对意料之中的结果心生不满，她把弹匣装了回去，站起身来。\
“……行吧，看样子也不是人人都有本事改装这玩意儿”\
\
刚才的课上用掉的弹药是两发。加上未使用的一枚，没有证据说明那个射手持有规则以上的弹药。\
\
“喂喂喂，你这是……”拉斯普京瞪大双眼，看着副会长把接触枪揣进了自己的口袋。\
\
“我保管起来，总比被随便什么人冒领走了强。”\
不知为何她感到自己像是暗暗松了口气，语调也随之轻快起来。\
“再说，我还有话要跟校医院说。我去还这个，他们总不能不让我进门儿吧？”\
\
“……唉。”\
脑袋露在被垮上肩头的书包之外，拉斯普京发出了“我可什么都不知道”的哀叹。“说实话，我现在有点儿后悔把魔法少女这事儿交给你了。”

\
“——然后我一看，那是个被丢到村庄外的尸体。好家伙，一看就是得了鲸瘟不治而死，脖子往上全是药草熏蒸留下的恶心黄色，已经腊入味儿了。再加上吸饱了土里的湿气，泡得面目全非。这么个东西被丢在路边那肯定是村里出了大事，我想赶紧过去看看，拉大车的老家伙从后面直拍我肩膀儿告诉我不用了。敢情那烟都已经——”\
高附讲起人生往事时眉飞色舞，和故事内容很不搭调。一扭头才发现供人休息的土坡上只剩下了自己和赭石两人，而他讲故事的对象并不在场。\
“嗯？她人呢？”\
\
赭石伸出一根兔爪来指了指天顶。\
\
“……多早晚去的啊？”\
\
“呃，大概是你讲到村里的地主在鲸瘟爆发前搬走了的时候吧”\
\
“……这孩子”高附的脸都皱成一团。“我就说嘛，对我讲的鲸瘟没反应，那除非是铁石心肠。”\
\
“……你干嘛要让她反应这个啊？”\
\
“对敌斗争可是非常严酷的。她这样的年轻人，要是没个深刻认识，迟早有被吓尿的时候”\
