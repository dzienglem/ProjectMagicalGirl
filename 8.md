“三发规则”有着漏洞。\
\
触发监督权需要条件。需要举出案例和疑点，才有可能使校医院出示射击记录、药剂使用记录其他检查结果。精英病是保密对象，为了防止走漏消息，校医院的出示内容并不会在校内公布。记录由学生会核对后，再向校内答复结论。\
\
换句话说，如果没有案例，学生会无法介入。但是制造一个满足条件的案例几乎毫无机会。事实上，精英病在校内蔓延已有数年，从未有过一届学生会就此过问校医院。

\
在人们不抱希望的时候，春雪最终还是下起来了。\
\
对副会长而言这时机并不很妙，因为她正在上的这堂早上的文学课上出现了群集发病。射手没能成功同时压制两名发作者，课堂被指示解散。校医院应召而来，中弹昏厥的患者被用担架抬走送往校医院，另一人在其他学生和教员撤退一空的教室里血流如注地独自发狂，形如丧尸。在他陷入过呼吸的空当，校医团队七手八脚，手持各自的镇暴钢叉一拥而上。患者被捉拿归案，在教室外待命的第二队校医抹了把汗，把已经捏在手里的室内用催泪弹放回随身小柜。教室隔壁楼梯拐角处，护士在搀扶着受伤的射手慢慢坐下——在他试图射击时，患者突然使用讲台上的花瓶反击。一番抵抗后飞出去的接触枪击碎了窗户。玻璃在两人至近距离爆裂，杂夹着雪花的冷风灌了一屋子——撤退指令就是在那之后发出的。\

“你说的没有选择是什么？”\

“他就剩一发阻断剂了，而又没有工夫给他等待时机。他只能接近那个人来射击”\

副会长用手肘怒怼了一把自己的书包，让里面的兔子吃了一记不轻不重的腹拳。尽管没有旁人，其实无需掩饰尴尬，她还是下意识抬起手来撩了撩腮边的发丝。\

“喔唷。那还真……真是意外。我还以为你们肯定谈过这件事儿呢。”\

“谈过是不假。但是，”\
“我没法料到她对三发规则本身怎么想。”\

“哇，这更意外了。不是，”白兔害怕再挨一个肘击，连忙辩解。“我是说啊，你和魔法少女的交情这么铁，你可以说是了解她的一切，却居然……？”\

“谁告诉你我了解一切啊？别净自己瞎脑补”\

“也许她试图做什么却又没告诉我呢？这样我不就没法知道了嘛。”\

副会长倏地停下脚步。拉斯普京张口结舌。\

副会长和朋友据理力争时，拉斯普京被按死在了口袋中，谈话的内容是精选后由副会长转达的。他已经了解到‘魔法少女’暗中出手，特赦罪人并解散调查组一事。\

“不是那个”\

“……你怎么这么啰嗦，我哪儿知道是哪个。”\

“好吧好吧。但你看啊，她既然有权力干涉调查，为什么不能修改规则呢？她是——唉，我忘了怎么说。兴许八成背不住就能给你们加几发子弹呢……！”\

“这根本不是加几发子弹的问题”\
再三确认校医已经全部离开，副会长重新迈进教室。破窗中吹来的风夹杂着雪花，在另一边也能感到寒气。\
“行，就算是三发再加三发。群集发作现在还只是冒了点儿烟，没火苗子。今天是俩人成一波，下次说不定是六个人？七个人？十个人？那时候怎么办？再加更多子弹？那这个规则不就是在闹着玩儿？”\

人们相信三发规则足够保护自己。——正因为它的存在，即使有精英病，校内也安全无事，一如既往。\
改变三发规则的尝试，最终都会动摇这个信念的根基。\

“……所以呢，你动它一个试试，你看人们不把这里房顶掀了”\

“按你说的，三发规则就是在你们这儿使用阻断剂的基石，”\

卸下来的书包被放在课桌上，里面的拉斯普京终于平稳着陆，可以慢条斯理地说话了。副会长在讲台侧的墙边跪下，弯下身来打开高度和腰平齐的通风百叶窗。\

“……但是，你不也在怀疑这个规则已经被人打破了吗？”\
“要不然，你也不会上这儿来。对不对？”\

刚才的课上用掉的弹药是两发。加上未使用的一枚，没有证据说明那个射手持有规则以上的弹药。\

“喂喂喂，你这是……”拉斯普京瞪大双眼，看着副会长把接触枪揣进了自己的口袋。\

“再说，我还有话要跟校医院说。我去还这个，他们总不能不让我进门儿吧？”\

“……唉。”\
脑袋露在被垮上肩头的书包之外，拉斯普京发出了“我可什么都不知道”的哀叹。“但愿你还没忘，咱们还有蹄鲸要对付，像这样对突发事件大包大揽可不是明智之举。”\

“急啥嘛，他都那么大个人了，丢不了”\
\
“那可是塞兰耶！本来就是笨蛋，走路都会掉到沟里，再说如今，他……”\
鬲昆欲言又止。\
“……反正你懂得，肯定没好事”\
\
“我懂什么呀，不就是他犯了那个花痴病吗？”\
吐出“花痴病”一词的高附不假思索，吓得从鬲昆背后跟上来的赭石当场石化。\
“那也未见得让他迷路。他指不定是在哪个把式家多喝了两碗鱼汤才拖得这么久”\
\
“就，赛兰他呀……”\
鬲昆少不更事，高附的话反而徒增了他的困惑。\
“他现在变得好恐怖哦。当面叫他笨蛋他都不生气了。”\
\
“等你长大就会懂了”\
高附直起身来，情不自禁拍了拍鬲昆的头。\
\
“花，花花花……？”\
赭石结结巴巴。\
\
“哇，你原来不是装傻吗？那你比鬲昆强不了多少啊，亏你还是大人”\
\
高附莫名兴致高昂。赭石仿佛脑子开锅一般从头顶升起热腾腾的蒸汽，鬲昆则是一脸困惑。在三只兔子身后，人类幽然开口了。\
“……这事儿你不要想想办法吗？”\
\
“你说得容易。天要下雨……！我能有什么法子好想？”\
\
高附回答得理直气壮。人类皱起了眉。\
“她是人类哎？”\
\
“这谁看不出来。可这种事情，哪里还顾得上对方是不是人类？这就像是，闪电认针，你在天上打个闪电，他就能借着那点儿光把线穿到针眼儿里去，就那么快”\
“要我说，这都要怪你。谁叫你那天晚上吓唬他？赛兰被枪口顶着都魂飞魄散了，这时候阻止你开枪并把他救下来的人，在他眼里就是天使。他因此对小姑娘一见钟情，不也非常自然吗？”\
\
“我最后不也没开枪吗？”\
高附的解释超出了人类的理解范围。“反正，他得醒醒了。他们不可能一直有机会见面。等鲸的问题处理完，到那个时候——”\
\
“知道知道。可这事儿吧，它得慢慢来。你不能趁赛兰最神魂颠倒的时候给他这种刺激，对不对？而且啊，”高附抓了抓毡帽下面的后脑勺。\
“别看赛兰这样，他还是个单纯小鬼呢。”\
\
“他，他来了”鬲昆捅了捅高附，然后立刻缩回他背后。在他紧盯不放的方向，赛兰率领的小队穿过连接洞外临时搭建起的栈道。
***
\
“……他们应该给这玩意儿贴个钢化膜啥的。粘得本来就不牢，材料么也就那样——压根焊不到一起，分分钟就会碎了。我说这可咋整？把式说她也没辙。钢箍、支架底座都得等收集到所有碎片后量尺寸现做，听了就让人头大——地下人也真是无药可救，本来就滑不拉几的东西还非要焊成个圆疙瘩，不是吃多了就是喝多了。”\
\
“好好好，你先把湿披风脱了再抱怨。再说，是让你跟把式打听这是什么东西，不是叫你把它修好，你没忘了吧？”\
赭石接过队长的毡帽，抖掉上面的雪片。——地上下着鹅毛大雪，而地下坑道温暖如春，赛兰的鲸鬃披风上的雪迅速融化，洇在下摆。用小爪子把插在腰上的号角拨拉到一边，赛兰摸出了贴身保管的、亮晶晶的蛋壳碎片。\
\
“嗯？可‘她’不是说……”\
赛兰环顾四周。\
“——呃，等等，咋就你一个人在这儿。”\
\
人类不知为何显得不必要地戒备森严。\
“——她太忙了，没法天天过来。这个你知道吧？”\
\
“知道知道。‘她’还是你们这些崽子的‘学生会长’，对吧？”\
“可既然咱们已经是盟友了，就算不是战时，也得通通气儿，说说知心话儿啥的不是？”\
\
“我也是盟友，你有什么知心话儿也都先跟我说吧。剑呢？”\
\
“我让俩弟兄在洞口守着呢。那玩意儿太沉了，上不了栈道。等会儿还得叫运输队送到工厂去”\
\
“哇，我是在做什么福报吗，一天天的，从晚上九点到早晨九点，连续看了六天场子，不光没人说一句知心话儿，队长还只想让我加班？？”\
高附装模作样地表示了委屈。\
“怎么着，把式觉得十拿九稳？我可不想再押着任何庞然大物走返程了。”\
\
“没事儿，拿去吧。”赛兰从递过来的小搪瓷缸里喝了一大口热茶。“咱都答应人家了，不能出尔反尔。再说自古以来……！地上的兔子就是草原上的雄鹰，不像地下的懦夫。既然成了盟友，她的……你们的问题就是我们的问题”他伸出拳头捶了捶自己的胸脯。“我说的这话，崽子你回头别忘了转达给她。”\
